# Demo Bank Application

## Features

- User Authentication and Registration
- Account Management(Balance Checking, Deposit, Withdrawal, Transfer)
- Deposit and withdrawal methods(Cash, Bank Transfer, Mobile  Money, cryptocurrency)
- Transaction History
- Real-time Currency Exchange
- Loan Management
- Card Management

this is a demo bank application that allows users to perform various banking operations such as deposit, withdrawal, transfer, and loan management. this should be a full-stack application with a user-friendly interface and secure authentication system built with python and django framework and tailwind css for the frontend.

### Tech Stack
- Python
- Django
- Tailwind CSS
- HTMX (for dynamic content and ajax requests to minimize page reloads)
- HTML
- CSS
- JavaScript
- Git
- GitHub
- Docker
- Google Cloud Run for immediate deployment
